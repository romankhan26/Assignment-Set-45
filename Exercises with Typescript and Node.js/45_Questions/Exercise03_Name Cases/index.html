Name Cases

In TypeScript, “name cases” often refer to the conventions used for naming variables, classes, interfaces, and other entities. Here are some common name cases used in TypeScript:

  • PascalCases
  • camelCases
  • UPPERCASE

However, if you’re referring to the manipulation of string cases in TypeScript, there are built-in methods to convert strings to different cases:

  • toLowerCase(): Converts a string to lowercase.
  • toUpperCase(): Converts a string to uppercase.
